What else should we add to Level 2?

The search bar + Groq results are a given. Pick any additional features that appeal to you. These are all pure sandbox enrichment — no new objectives required.

A

Guestbook

The VintageArchive or ValleyComputer page has a "Sign our Guestbook!" link. Clicking it opens a classic Netscape form modal — Name, Location, Message, Submit. On submit, a new entry appears at the top of the guestbook. Pure 90s social ritual.

B

Download Dialog

A "Download Free Screensaver" or "Get Netscape Now!" link on one of the pages triggers a faithful Win98 download dialog — filename, file size, estimated time, progress bar that fills over ~3 seconds, then a "Download Complete" state. No file actually downloads.

C

Dial-up Modem Sound

On the initial boot/loading sequence and on each page navigation, play an authentic dial-up modem sound effect (the squealing handshake tone). Short version (~2s) for page loads, full version (~5s) for the initial connection. Sound can be toggled off in the status bar.

D

View Source

The browser's View menu → "Source" (or Ctrl+U) opens a new window showing the raw HTML of the current page — formatted like Netscape's view-source, monospaced, light gray background. Shows real-ish 1995 HTML with table layouts, font tags, and bgcolor attributes. A nod to how people learned web dev in 1995.

Select as many as you like, then let me know in the terminal.
